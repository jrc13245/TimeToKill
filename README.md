Features:
TimeToKill - cut down version of EliteWarriorCombat by jlabranche, improvements and superwow support by Torio/jrc13245

shift-drag when unlocked to move - clickthrough enabled when locked
  
 /ttk
 
 /ttk name on|off

 /ttk combathide on|off
 
 /ttk unlock|lock

 /ttk smooth <0.1-0.5> (default 0.3)

 /ttk conservative <0.9-1.0> (default 0.95)
 
 /ttk minsample <seconds> (default 2.0)
 
 /ttk burst <multiplier> (default 3.0)
 
 /ttk combatlog on|off
 
 /ttk status
 
 /ttk debug
